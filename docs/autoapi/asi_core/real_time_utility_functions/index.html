asi_core.real_time_utility_functions.parse_logger_data(url_cs_logger_table, timezone, name_desired_columns_cs_table=None)¶

This function access the table of a campbell scientific logger via its URL. All channels and the time stamp are retrieved from the HTML string and placed in a data frame. If desired some channels can be renamed. :param url_cs_logger_table: (str) url of table as string :param timezone: (str) desired time zone (e.g. “GMT+1” = UTC+1) :param name_desired_columns_cs_table: (dict) List with 2 columns and n rows. First column holds the original names of channels

which shall be renamed. The second column holds the new names for the channels.

Return df:

data frame single row with n columns for each channel + timestamp index as datetime
